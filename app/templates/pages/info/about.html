Our Mission: Organize Your Thoughts into Actions

Welcome to Projectify, a modern productivity app built on a simple premise: your life isn't one-dimensional, so your tools shouldn't be either.

We all juggle different roles. You might be a student, an athlete, a creative, and a meticulous planner all at the same time. Traditional productivity apps force all of these contexts into a single, messy list. Projectify is designed to fix that.

What Makes Projectify Different? The "Identities" System

The core of Projectify is the Identity. Think of an identity like a workspace. An identity is a "hat" you wear in life. Instead of one cluttered dashboard, you create a distinct space for each of your roles:

  • The Learner: For tracking educational goals and new skills.
  • The Builder: For managing your side projects and tasks.
  • The Health Nut: For logging workouts and wellness goals.
  • The Organizer: For handling schedules, budgets, and daily life.

By switching your active identity, you change your entire context. You only see the projects, tasks, and thoughts relevant to that role, allowing you to focus completely on what matters most in the moment. Customise the names of the identities and make them yours in the Settings menu.

Features at a Glance

Identity-Based Management:
A unique system to segment your projects, tasks, and thoughts by your different roles.
Atomic Project Status:
A project's status (Not Started, In Progress, Completed) is automatically calculated based on the state of its tasks. You manage the work, and Projectify manages the status.
Thoughts Journal:
A "chat-style" journal for each identity with a filterable timeline, allowing you to capture ideas and reflections in the right context.
Dynamic Theme Engine:
A powerful, database-driven theme system lets you choose from multiple color schemes and toggle between Light, Dark, and System modes.
Modern API-Driven UI:
Built with a fast, modern frontend that uses asynchronous JavaScript for all task and thought management, so the page never has to reload.

About the Developer

Hi, I'm Alan O'Connor, the developer behind Projectify. I built this application as a way to explore the challenges of building a modern, full-stack web application from the ground up.

This project was a deep dive into building a robust, scalable backend architecture using Flask, combined with a responsive, intuitive, and beautiful frontend. My focus was on writing clean, reusable code (like the Manager pattern) and ensuring all database operations are atomic and secure.

Built With

Projectify is built with a professional-grade Flask and PostgreSQL stack.

Backend:
Flask, using an Application Factory pattern and Blueprints.
Database:
PostgreSQL with Flask-SQLAlchemy and Flask-Migrate (including native enums).
Architecture:
A custom-built, atomic Manager Class architecture to keep business logic out of routes.
Authentication:
Flask-Login with bcrypt hashing and Flask-WTF for CSRF protection.
Frontend:
Jinja2, JavaScript (ES6+), and a custom HSL-based CSS variable theme engine.
